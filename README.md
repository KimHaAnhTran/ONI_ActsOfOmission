# O.N.I. Acts of Omission

---
# About
A U.S. history simulation game where you play as a typist for the Office of Naval Intelligence, circa 1890-1929.
Transcribe documents on a typewriter and learn about the early stages of American imperialism.
A project for AP Capstone Research.
